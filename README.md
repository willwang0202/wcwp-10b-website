# 🎤 Put On a Show — Budget Game

An interactive web-based budget game for **WCWP 10B** where you plan a live music show under real financial constraints.

## 🎮 How It Works

1. **Choose your artist identity** — Working Class ($500), Successful ($3,000), or Trust Fund ($10,000).
2. **Allocate your budget** across categories like Venue, Gear, Promotion, and Staff.
3. **See your results** — find out if your show is a hit, a hustle, or a disaster.

Every choice affects your total spend *and* the time you invest. Can you put on a great show without going broke?

## 🛠 Tech Stack

- **HTML** / **CSS** / **JavaScript** — no frameworks, no build step
- Google Fonts (DM Serif Display + Inter)
